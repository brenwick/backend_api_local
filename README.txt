Download From Github: https://github.com/brenwick/backend_api_local


To Run Local Server:

1) Download Node.js and NPM (node package manager installed by default)
    https://nodejs.org/en/download/

2) Download directory from Github and navigate to the directory from CLI 

3) Run the command "npm install" on the base directory to download all dependencies

4) Run the command "node server.js" to start the local server

5) Now, the server should be listening on "localhost:3050"

6) Send HTTP requests to localhost:3050 and check your terminal for this debug information:
    HTTP Headers
    Body
    Status Code
