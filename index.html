Welcome to the backend API

Each route expects a certain JSON input and will deliver a JSON output. To test them, please append the route name to the end of the URL.

Example: https://screenlimit-backend-api.herokuapp.com/api/profile

/api/profile

Sample Input:

{"username" : "Brandon", "email" : "brandon@gmail.com", "phone" : "123-456-7890"}

Sample Output:

{"message" : "user: Brandon has email: brandon@gmail.com and phone #: 123-456-7890"}

/api/string

Sample Input:

{"string" : "helloworld"}

Sample Output:

{"message" : "Here is your string: helloworld"
